# Coleções e Streams em Java

Este repositório contém exercícios práticos desenvolvidos durante o estudo de **Collections e Streams em Java**.

O objetivo do projeto é praticar manipulação de dados utilizando as principais estruturas de coleções da linguagem Java e a API de Streams.

## Conteúdos praticados

- List
- Set
- Map
- Streams
- filter()
- map()
- reduce()
- min() e max()

## Estrutura do projeto

##  Como executar o projeto

1. Compile os arquivos Java:


javac src/*.java


2. Execute o programa principal:


java -cp src Main


O arquivo `Main.java` executa todos os exercícios em sequência.
